# Example: Leiden Longevity Study

A minimal worked example of a static FDP, originally from the
[EJP-RD FDP](https://w3id.org/ejp-rd/fairdatapoints/wp13).

Contains a `dcat:Catalog` with two `dcat:Dataset` entries:

- **Blood Cell Counts Nonagerian Siblings** — hematological data from the LLS
- **Demography** — demographic data from the LLS

These files illustrate the DCAT structure that a static FDP serves.
Use them as a reference when creating your own catalog and dataset metadata.
